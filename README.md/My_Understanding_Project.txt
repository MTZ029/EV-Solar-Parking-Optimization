EV charging session Dataset
The dataset has 8530 charging sessions and 11 variables describing the behavior of the electric vehicles arriving
at the parking lot.

| Variable                | Description                                    |
| ----------------------- | ---------------------------------------------- |
| day                     | Day index of the year                          |
| vehicles_generated_day  | Total number of EVs generated on that day      |
| session                 | Charging session identifier                    |
| assignment              | Parking/charger assignment identifier          |
| connection_minute       | Arrival time of the EV (minutes from midnight) |
| full_minute             | Time at which the vehicle reaches full charge  |
| disconnection_minute    | Departure time of the EV                       |
| connection_timestamp    | Arrival timestamp                              |
| full_timestamp          | Timestamp when charging is completed           |
| disconnection_timestamp | Departure timestamp                            |
| energy_kwh              | Total charging energy requested by the EV      |

Each charging session includes energy demand values ( enrgy_kWh )

The data analysis ~ Parking durations
-------------------------------------
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The parking duration = disconnection_minute - connection_minute
converted from minutes to hours.

The results:
Most EVs remain parked for several hours, mainly between 6h to 10h   ~ Chraging demands are flexible 

So what we want is that, instead immediately after the arival, car doesn't need to charge, we can shift it.

In the optimization problem, each EV will have a constraint of the form:

connection_time <= charging_time <= disconnection_time

and the total delivered energy must satisfy the required session energy.

# flexibility analysis:
----------------------
||||||||||||||||||||||
flexibility = parking duration - charging duration

The majority of EV charging sessions possess several hours of scheduling flexibility, this means that charging 
does not necessarily need to begin immediately after vehicle arrival. ~ As by the supervisor.


Instead, charging power can be shifted within the available flexibility window to:

- maximize self-consumption of solar PV generation,
- reduce peak power demand,
- reduce electricity costs,
- improve overall parking lot energy management.


Donc for most vehicle we have 1 - 5 hours of flexibility, ~ the time stamps.
