Main conclusion:
Flexible EV charging provides small but consistent economic benefits compared with ASAP charging.
The benefit is limited because the optimally sized PV+BESS system already handles much of the demand
effectively.

For the medium-demand day, flexible charging reduces energy cost but increases peak grid demand. This shows
that the current objective is cost-oriented and not peak-oriented. If peak shaving is required, an
additional peak constraint or stronger peak penalty should be included.
